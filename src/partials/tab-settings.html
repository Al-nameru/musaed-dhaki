🌐 إعدادات عامة

يعرض هذا القسم عرضاً شاملاً لجميع إعدادات الصفحة دون تخصيص لمجال واحد فقط. يمكنك الوصول إلى الضبط العام لمزيد من التحكم في سلوك التطبيق وسهولة التخصيص.

هذا التبويب يجمع رؤية عامة لجميع الإعدادات المتاحة داخل صفحة الإعدادات، بما في ذلك:
  • ⌨️ الاختصارات العالمية
  • 🔊 إعدادات النطق وصوت القارئ
  • 🤖 إعدادات النماذج المحلية
  • 🎙️ محركات تحويل الصوت إلى نص
  • ⚙️ سلوك معالجة النصوص
  • ✍️ تخصيص التوجيهات والبرومبت

كيف تستخدم هذا التبويب

هذا القسم مخصّص لعرض الإعدادات العامة في مكان واحد وتسريع التنقل بينها. إذا أردت تعديل إعداد محدد، انتقل إلى التبويب الخاص به أو استعرض المحتوى المباشر في هذه الصفحة.

⌨️ الاختصارات العالمية (Global Hotkeys)

تتيح لك هذه الاختصارات تشغيل المساعد في أي وقت بالخلفية. اختر الاختصارات المفضلة لديك ثم اضغط حفظ:
يدعم زرًا واحدًا أو تركيبة من Ctrl / Shift / Alt مع زر آخر
الاختصار النشط حالياً: Ctrl + Shift + 1
يمكن استخدام زر واحد أو تركيبة من زرين أو ثلاثة أو أربعة
الاختصار النشط حالياً: Ctrl + Shift + 2

📤 تصدير نسخة احتياطية مشفرة

سيتم تشفير كامل إعداداتك ومفاتيح الـ API المدخلة باستخدام خوارزمية AES-256-GCM بكلمة المرور المحددة.

📥 استيراد نسخة احتياطية

اختر ملف النسخة الاحتياطية المشفر وأدخل كلمة المرور لفك التشفير واستعادة كامل بياناتك.

🔍 مفتش العناصر ( UI Element Picker )

أداة تفاعلية تتيح لك النقر على أي عنصر داخل واجهة التطبيق وتظليله والحصول على معرّفه الفرعي ( CSS Selector ) لنسخه مباشرة.

🎯 وجهة الإدراج التلقائي داخل التطبيق

حدد صندوق النصوص المستهدف لإدراج النصوص فور تسجيلها أو معالجتها داخل واجهة المساعد.
